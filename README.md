# pbir.tools -- Private Beta

pbir.tools is an agent-first toolkit for Power BI report manipulation.
It lets you read, modify, create, and validate Power BI reports (PBIR format)
through a CLI, Python object model, and MCP server -- without opening Power BI Desktop.

## What to expect

- Access to a private repo with installable binaries (Windows and macOS)
- Beta version software that's likely to have bugs
- A feedback channel to report bugs and request features
- Updates as new versions are published
- The private beta is expected to last a maximum of two weeks and a minimum of one week

## How to sign up

[**Sign up here**](https://github.com/data-goblin/pbir.tools-private-beta/issues/new?template=beta-signup.yml) by filling out the beta signup form.
If approved, you'll receive a GitHub invitation granting read-only access
to the distribution repository.

## Privacy

We only store your GitHub username for access management.
You can withdraw consent at any time by closing your signup issue.
